【このフォルダについて】

このフォルダには、
テストを実行する前に「使う環境」を切り替えるための
バッチファイル（ダブルクリックで動くファイル）が入っています。

※ この操作だけでは、テストは実行されません。
※ あくまで「準備（環境の切替）」です。


【使い方（重要）】

1. このフォルダを開きます
2. 目的に合った .bat ファイルをダブルクリックします
3. 黒い画面（コマンド画面）が表示されます
4. 「エラー」と書かれていなければ成功です
5. Enterキーを押して画面を閉じます


【どのファイルを使えばいいか】

・switch_env_internet_html.bat
　→ インターネット接続環境で
　　HTML形式のナレッジを使う場合

・switch_env_internet_markdown.bat
　→ インターネット接続環境で
　　Markdown形式のナレッジを使う場合


【注意事項】

・この操作は「環境の切替」だけを行います
・テストの実行は、別の手順で行ってください
・途中でエラーが表示された場合は、
　無理に続けず、担当者に連絡してください
・内容が分からない場合は、ファイルを編集しないでください


【補足】

この操作によって、
「どの条件でテストを行ったか」を
後から正しく確認できるようになります。

これは、結果の信頼性を保つために
とても重要な手順です。
