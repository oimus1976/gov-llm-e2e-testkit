# F4 v0.2 Trial Dataset（試金石データ）

本ディレクトリは、
**F4 v0.2 における試験結果を
「判断可能な試金石データ」として提供するための成果物一式**
である。

---

## 1. このデータの目的

- HTML / Markdown という RAG 入力差分が
  Qommons.AI の応答に与える影響を
  **誤解なく観測・比較する**
- 他プロジェクトが
  **採否判断の根拠として再利用できる**
  データを提供する

本データは、
**結論を示すものではない**。

---

## 2. データ構成

本データは、以下の三層で構成される。

| 層 | 内容 |
|---|---|
| Raw | モデルの生回答（一次証拠） |
| Context | 実行条件・前提 |
| Summary | 人間向け差分整理 |

---

## 3. 利用上の注意

- summary のみを見て判断しないこと
- raw データを正本とすること
- SKIP（Inconclusive）は失敗ではない
